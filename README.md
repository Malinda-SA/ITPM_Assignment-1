This repository contains the automation scripts for *ITPM Assignment 1*. 
The project utilizes *Playwright* to test the functional and UI aspects of the SwiftTranslator web application.

## Project Setup

### Prerequisites
1. Ensure *Node.js* is installed on your machine.
2. Ensure you have an active internet connection.

### Installation Steps
1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Install the necessary dependencies by running:
